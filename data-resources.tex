
\section{Data and Resources}

Additional calculations and figures were done using MATLAB (\url{http://www.mathworks.com}, last accessed April 2018). Map figures were prepared using the Generic Mapping Tools, GMT (\url{http://gmt.soest.hawaii.edu/}, last accessed April 2018).
